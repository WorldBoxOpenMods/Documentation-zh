# 介绍

由于你游模组作者UI都写得不太行, 反观BepInEx的菜单能够自动布局, 因此特别推出这个自动布局窗口, 你只需要简单地开启(垂直/水平/网格)分组, 然后加入设定大小的组件即可自动布局.
